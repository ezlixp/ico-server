# ico-server

## About

This is the web api behind the [guild api](https://modrinth.com/mod/guild-api) minecraft mod and the [ogybot](https://github.com/zpdh/ogybot-discord) discord bot.
This project is mainly maintained by me, but feel free to
open pull requests to merge new features, bug fixes, improvements, etc.

## Contributions

Contributions are heavily appreciated, however, before you commit please make sure the code is readable and doesn't
break anything.

It is recommended that you develop from the [combined repository](https://github.com/ezlixp/wynn-bridge.git). This repository includes a compose.yaml file to run both the bot and the web api together using Docker Compose, and vscode launch profiles for debugging. For further information, check the README in that repository.

Thanks for checking out my little project!
